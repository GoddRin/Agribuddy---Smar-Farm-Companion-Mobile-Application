# AgriBuddy - Smart Farm Companion 🌾

AgriBuddy is a comprehensive decision support and farm management system specifically designed for farmers. It offers intelligent crop tracking, real-time analytics, and automated scheduling to help optimize farm operations and increase yields.

Built for the modern farmer, AgriBuddy simplifies complex data into actionable insights through its intuitive dashboard and interactive companion, Mang Pedro.

## 🚀 Key Features

- **Smart Dashboard:** At-a-glance overview of farm metrics including productivity trends and crop health.
- **Crop Management:** Detailed tracking of crop stages, health status, and growth cycles across different blocks.
- **Intelligent Expert (Mang Pedro):** A 24/7 expert system that provides guidance on irrigation, pest control, and general farming knowledge.
- **Interactive Logs & Tasks:** Effortless logging of field observations and automated task reminders for daily farm maintenance.
- **Financial Analytics:** Track farm expenses and projected returns with visual charts and reports.
- **Multilingual Support:** Fully inclusive with support for Tagalog, English, and Taglish users.

## 🛠️ Tech Stack

- **Framework:** Flutter (Mobile & Web)
- **State Management:** Riverpod
- **Animation:** Flutter Animate & Lucide Icons
- **Expert Reasoning:** Large Language Model (Llama 3.3) integration for specialized agricultural reasoning
- **Platform Support:** Android (APK) and Web (Vercel)

## 🛠️ Setting Up (Development)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GoddRin/Agribuddy---Smar-Farm-Companion-Mobile-Application.git
   ```
2. **Install Dependencies:**
   ```bash
   flutter pub get
   ```
3. **Configure Environment:**
   Create a `.env` file in the root directory and add your API credentials:
   ```env
   GROQ_API_KEY=your_api_key_here
   ```
4. **Run the App:**
   ```bash
   flutter run
   ```
